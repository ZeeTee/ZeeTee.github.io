scrapy基础概念-命令行工具

scrapy是通过命令行中一行行的命令进行控制的。这里包含很多命令。

scrapy项目的目录结构

首先我们需要了解scrapy项目的目录结构。每一个scrapy项目都有相同的默认目录结构：

projectname/
    scrapy.cfg
    projectname/
        __init__.py
        items.py
        pipelines.py
        spiders/
            __init__.py
            spider1.py
            spider2.py
            ...

其中，scrapy.cfg所在的目录为项目的根目，该文件是项目的配置文件，其中配置了setting文件的路径。

使用命令行工具

在命令行中敲入scrapy不加任何的参数，可以看到使用帮助和可使用的命令。

(py34) mbp:gitdir zhang$ scrapy
Scrapy 1.3.2 - no active project
# 若命令在项目文件内运行结果为：
# Scrapy 1.3.2 - project: quotesbot

Usage:
    scrapy <command> [options] [args]

Available commands:
    bench         Run quick benchmark test
    commands      
    fetch         Fetch a URL using the Scrapy downloader
    genspider     Generate new spider using pre-defined templates
    runspider     Run a self-contained spider (without creating a project)
    settings      Get settings values
    shell         Interactive scraping console
    startproject  Create new project
    version       Print Scrapy version
    view          Open URL in browser, as seen by Scrapy

    [ more ]      More commands available when run from project directory

Use "scrapy <command> -h" to see more info about a command

创建项目

当你想创建一个scrapy项目时使用：

scrapy startproject projectname [project_dir]

这个命令会在project_dir目录下创建项目projectname。project_dir为可选参数，若没有给出，project_dir和projectname相同。
