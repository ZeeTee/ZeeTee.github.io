Git学习

Git使用

新建仓库

使用命令git init,可以将当前目录建成一个空的仓库。

添加文件

添加文件的过程分为两步：

  1. 使用git add <file>添加文件，该过程可以重复多次。
  2. 使用git commit -m <message>提交改动到————，并包含提交内容的改动信息。

文件修改后同样是通过这两步将文件提交到仓库保存。其中当需要查看两个文件的差异时可以使用git diff查看修改内容。

掌握Git状态

使用命令git status,查看工作区(working tree)状态。

返回前一版本

当你想要返回你当前版本的前一版本时，首先需要查看提交历史记录：git log。使用git reflog可以查看所有的命令历史。

现在确定返回上一版本之后，使用命令git reset --hard HEAD^。将HEAD^更换为commit id可以跳到指定的提交版本(可以用来实现向前跳跃)。
